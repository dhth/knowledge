Project root
---

For a directory structure like:

```
.
├── .gitignore
├── .vim
│   └── coc-settings.json
├── project
│   ├── app
│   │   ├── __init__.py
│   │   ├── config.py
│   │   └── main.py
│   └── requirements.txt
└── pyrightconfig.json
```

`pyrightconfig.json` would need:

```json
"executionEnvironments": [
    {
        "root": "project"
    }
]
```
