Run Script
===

Use emojis in AppleScript
---

`BOOKMARK_TITLE` is a workflow variable.

```AppleScript
set bookmarkTitle to do shell script "echo " & quoted form of (system attribute "BOOKMARK_TITLE") & " | iconv -f UTF-8-MAC -t MACROMAN"
```
