## `%mprun`

`mprun` works only for functions defined in separate modules rather than the notebook itself.

```python
%%file mprun_demo.py
def sum_of_lists(N):
    total = 0
    for i in range(5):
        L = [:fontawesome-solid-link: j ^ (j >> i) for j in range(N)]
        total += sum(L)
        del L # remove reference to L
    return total
```
